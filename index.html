{domain}/datasets/add

Add a custom dataset.

Authorizations:
basicAuth
query Parameters

required
string

Name for new dataset.

task
required
string

Inference task intended for new dataset.

path_location
required
string

URI to an external dataset. Send empty if uploading private dataset.

secret_id
required
string

Secret ID associated for external dataset. Send empty if uploading private dataset.

string

{domain}/models/predict

Make an inference against a deployed model.

Authorizations:
basicAuth
query Parameters
model_id
required
string

Model ID associated to the model for prediction.

task
required
string

Inference task for the model.

  • If task is 'custom', request body must contain an 'instruction' key/value.
  • If task is 'classification', 'embeddings', 'named_entity_recognition', 'question-answering', or 'summarization', request body must contain a 'text' key/value.
  • If task is 'question_answering_with_context', request body must contain 'question' and 'context' key/value's.
  • If task is 'question_answering_no_context', request body must contain a 'question' key/value.
  • If task is 'machine_translation', request body must contain 'text' and 'context' key/value's.
Request Body schema: application/json

Prediction inputs.

text
required
string

Leave blank except for inference task types ['classification', 'embeddings', 'named_entity_recognition', 'question-answering', 'summarization', 'machine_translation'].

question
required
string

Leave blank except for inference task types ['question_answering_with_context','question_answering_no_context'].

context
required
string

Leave blank except for inference task types ['question_answering_with_context','machine_translation'].

instruction
required
string

Leave blank except for inference task type 'custom'.

Responses

https://api.bookend.ai/{domain}/models/predict
